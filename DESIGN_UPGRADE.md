# TrendRadar 界面设计升级说明

## 设计灵感

本次界面升级参考了 [wangmazha.com](https://wangmazha.com/) 的现代化设计风格，旨在提供更清晰、更美观的用户体验。

## 主要改进

### 1. 配色方案升级

**旧版本：**
- 头部渐变：`#4f46e5` → `#7c3aed`（紫色双色渐变）
- 背景色：`#fafafa`（浅灰）
- 容器宽度：`600px`

**新版本：**
- 头部渐变：`#667eea` → `#764ba2` → `#f093fb`（紫色三色渐变，更丰富）
- 背景色：`#f5f7fa`（更淡雅的灰蓝）
- 容器宽度：`820px`（增加阅读空间）

### 2. 卡片式设计

**新增特性：**
- 每个关键词组采用独立白色卡片设计
- 卡片带有柔和阴影：`0 2px 8px rgba(0, 0, 0, 0.04)`
- Hover 效果：卡片上浮 + 阴影加深
- 圆角从 `12px` 增加到 `16px`

### 3. 标签系统优化

**渐变标签：**
- 热门词汇（≥10条）：红色渐变 `#ef4444` → `#dc2626`
- 温热词汇（≥5条）：橙色渐变 `#f59e0b` → `#d97706`
- 关键词标签：紫色渐变 `#8b5cf6` → `#7c3aed`
- 排名徽章：不同等级使用不同颜色渐变
- RSS内容：绿色主题 `#10b981` → `#059669`

### 4. 字体优化

**新增字体：**
- 引入 Google Fonts 的 `Noto Sans SC`
- 更好的中文显示效果
- 字重层次更丰富（400、500、600、700）

### 5. 视觉层次增强

**渐变文字效果：**
- 关键词标题使用渐变色
- Footer 项目名称使用渐变色
- 使用 `-webkit-background-clip: text` 实现

### 6. 交互体验提升

**Hover 效果：**
- 新闻卡片 hover 时右移 4px
- 按钮 hover 时上浮 2px
- 链接 hover 时颜色变为紫色
- 所有过渡动画采用 `ease` 或 `cubic-bezier` 缓动

### 7. 头部信息优化

**新增元素：**
- 副标题："追踪热点，洞察趋势"
- 信息卡片采用玻璃态效果（半透明 + 模糊背景）
- 4列网格布局展示关键信息

### 8. 新闻项设计改进

**视觉优化：**
- 新闻项背景改为浅灰色卡片（`#f9fafb`）
- 序号改为带边框的圆角方块
- 来源标签、排名、时间等使用不同颜色区分
- NEW 标签使用橙色渐变 + 阴影

### 9. 响应式优化

**断点设置：**
- `768px`：平板设备适配
  - 头部信息改为 2x2 网格
  - 调整内边距和字体大小
- `480px`：手机设备适配
  - 头部信息改为单列
  - 保存按钮改为纵向排列

### 10. RSS 部分独立样式

**视觉区分：**
- 使用绿色主题与热榜区分
- 独立的卡片容器
- 左侧绿色边框标识
- Hover 时背景色加深

## 技术细节

### CSS 改进

1. **现代化阴影系统**
   ```css
   box-shadow: 0 2px 8px rgba(0, 0, 0, 0.04), 0 1px 2px rgba(0, 0, 0, 0.06);
   ```

2. **渐变色定义**
   ```css
   background: linear-gradient(135deg, #667eea 0%, #764ba2 50%, #f093fb 100%);
   ```

3. **玻璃态效果**
   ```css
   background: rgba(255, 255, 255, 0.15);
   backdrop-filter: blur(10px);
   ```

4. **平滑过渡**
   ```css
   transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
   ```

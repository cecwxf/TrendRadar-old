# 市场Dashboard界面设计升级说明

## 设计灵感

本次界面升级参考了 [wangmazha.com](https://wangmazha.com/) 的现代化设计风格，将原来的深色主题改造为清新明亮的亮色主题。

## 主要改进

### 1. 主题配色大转变

**旧版本（深色主题）：**
- 背景色：`#0f172a`（深蓝黑）
- 卡片背景：`#334155`（深灰）
- 头部渐变：`#1e3a8a` → `#7c3aed`（深蓝到紫色）

**新版本（亮色主题）：**
- 背景色：`#f5f7fa`（清新浅灰）
- 卡片背景：`white`（纯白）
- 头部渐变：`#667eea` → `#764ba2` → `#f093fb`（紫色三色渐变）

### 2. 卡片系统升级

**新增特性：**
- 白色卡片 + 柔和阴影：`0 2px 8px rgba(0, 0, 0, 0.04)`
- 圆角从 `10px` 增加到 `16px`
- Hover 效果：上浮 4px + 紫色边框 + 加深阴影
- 卡片最小宽度：`300px`（原 `280px`）
- 内边距增加：`24px`（原 `20px`）

### 3. 渐变标签系统

**新的标签设计：**
- **交易所标签**：紫色渐变 `#8b5cf6` → `#7c3aed` + 白色文字 + 阴影
- **涨幅标签**：绿色渐变 `#10b981` → `#059669` + 白色文字 + 阴影
- **跌幅标签**：红色渐变 `#ef4444` → `#dc2626` + 白色文字 + 阴影
- **成交量标签**：浅灰背景 `#f9fafb` + 深灰文字

### 4. 字体与排版

**引入专业字体：**
- 中文字体：`Noto Sans SC`（Google Fonts）
- 字重支持：400、500、600、700
- 标题使用渐变文字效果
- 符号/代码也使用渐变色

### 5. 图表现代化

**ECharts 配色升级：**
- 图表背景：白色卡片 + 圆角 + 阴影
- 线条颜色：紫色 `#667eea`
- 区域填充：紫色渐变（0.25 → 0.02 透明度）
- Tooltip：白色背景 + 紫色边框 + 阴影
- 坐标轴：浅灰线条 + 深灰文字
- 网格线：浅灰 `#f3f4f6`

### 6. 时间范围选择器

**新设计：**
- 容器：白色卡片背景
- 标签：灰色文字 + 圆角背景
- Hover：浅灰背景 `#f3f4f6`
- 激活态：紫色渐变 + 白色文字 + 阴影

### 7. 分类标题优化

**视觉改进：**
- 主标题：24px + 渐变文字效果
- 分类标题：20px + 深灰色 + 浅灰下划线
- Emoji + 文字组合展示
- 更大的间距（20px）

### 8. AI 分析区域

**保持亮点：**
- 紫色渐变背景（更突出）
- 白色文字
- 更大的内边距 `32px`
- 增强的阴影效果

### 9. RSS 推文区域

**新设计：**
- 白色卡片背景
- 单个推文：浅灰背景 `#f9fafb`
- 左侧绿色边框 `#10b981`
- Hover 时背景加深 + 右移 4px
- 链接颜色：深灰 → Hover 时变紫色

### 10. 响应式增强

**移动端优化：**
- 容器无内边距（全屏显示）
- 头部圆角移除
- 卡片网格改为单列
- 图表高度减少到 300px
- 字体大小适配小屏幕

## 技术细节

### CSS 特性使用

1. **渐变文字**
   ```css
   background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
   -webkit-background-clip: text;
   -webkit-text-fill-color: transparent;
   background-clip: text;
   ```

2. **柔和阴影**
   ```css
   box-shadow: 0 2px 8px rgba(0, 0, 0, 0.04), 0 1px 2px rgba(0, 0, 0, 0.06);
   ```

3. **渐变按钮**
   ```css
   background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
   box-shadow: 0 4px 12px rgba(102, 126, 234, 0.3);
   ```

4. **平滑过渡**
   ```css
   transition: all 0.3s ease;
   ```
